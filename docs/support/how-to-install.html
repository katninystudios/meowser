How to Install Meowser

Learn how to install Meowser on available platforms.

Questions? Don't be afraid to ask on our GitHub!

This will teach you how to install Meowser on Windows, Mac or Linux.

info Installations for Android, Mac and Linux are not yet available ☹️. If you want to try Meowser anyways, please build it yourself (don't worry, it's not too hard).


Install from GitHub (Recommended)

Windows

  • Go to our GitHub Releases page
  • Download the .exe file (e.g. "Meowser-Setup-0.0.1.exe")
  • Run the .exe file from wherever you installed Meowser
  • Once installed, Meowser should automatically launch!

    • Enjoy your new browser!
